# CHANGELOG

## v2.6.0 - 2026-03-22

### 新增功能

- **Agent 攻擊系統 V2 升級**:
  - 攻擊手法分為 3 大類：語言變異類、隱喻影射類、社交黑話類 (移除技術對抗類)。
  - 引入 4 個能力維度：語言變異度、歷史文化厚度、圈層專業度、社會心理操縱度。
  - 新增 72 個 Agent，分為 6 組 (每組 12 人)：G1史鑑組、G2黑話組、G3阴阳组、G4同音组、G5反串组、G6暗流组。
  - 每個 Agent 根據人設分配擅長的攻擊手法和能力。
- **前端資料投喂窗口**:
  - 在網頁左側新增「資料投喂」窗口，用戶可手動投喂攻擊材料、黑話詞典、繞過案例給 Agent 學習。
  - 新增 `/agent/feed` API 接口，用於接收前端投喂的資料並更新 Agent 知識庫。
- **版本號顯示**:
  - 網頁上顯示當前版本號 `v2.6.0`。

### 改进与优化

- **代碼結構調整**:
  - 移除 `attack_knowledge.py`，引入 `attack_knowledge_v2.py` 以支持新的攻擊手法和能力維度定義。
  - `agents.py` 已全面更新，以適應新的 Agent 生成邏輯和攻擊系統。
  - `web_app.py` 已更新，以整合新的 Agent 系統和前端功能。
- **前端界面優化**:
  - 調整了前端佈局，確保「資料投喂」窗口與現有組件協調顯示。

### 其他

- 更新了 `README.md`，反映 `v2.6.0` 的所有變更。
